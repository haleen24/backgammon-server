## Регистрация и Авторизация

### Регистрация

Создает пользователя в бд
Адрес: ```/create-user```\
Тело:

```
{
"username": string,
"password": string
}
```

### Авторизация:

Авторизация добавляет токен в куки\
Адрес: ```/login2```\
Тело:

```
{
"username": string,
"password": string
}
```
